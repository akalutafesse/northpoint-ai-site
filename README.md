# North Point AI — Website

Static marketing site for [North Point AI](https://www.northpoint-ai.com).

## Development

No build step required. Open `index.html` in a browser or use any static file server:

```bash
python3 -m http.server 8080
```

## Deployment

Deployed automatically to GitHub Pages on push to `main`.

## Structure

```
index.html              — single page
assets/css/style.css    — styles
assets/js/main.js       — reveal animations
assets/img/             — logos, marks, OG image
```
